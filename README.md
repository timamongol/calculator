# RPN Calculator

Командный калькулятор, вычисляющий математические выражения с использованием обратной польской записи.

# Сборка

## Требования:
- CMake (версия 3.10+)
- Компилятор C++17

## Сборка проекта:

mkdir build
cd build
cmake .. -DCMAKE_BUILD_TYPE=Release
cmake --build . --config Release
